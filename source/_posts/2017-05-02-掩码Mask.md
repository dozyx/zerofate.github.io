[wiki Mask (computing)](https://en.wikipedia.org/wiki/Mask_(computing))



mask 是用于按位运算的数据。

常见的位掩码函数：

+ 使用`OR`置1

  ```
      10010101   10100101
   OR 11110000   11110000
    = 11110101   11110101
  ```

+ 使用`AND`值0

  ```
      10010101   10100101
  AND 00001111   00001111
    = 00000101   00000101
  ```

+ 检索某一位的状态

  ```
      10011101   10010101
  AND 00001000   00001000
    = 00001000   00000000
  ```

+ 反转位的值

  ```
      10011101   10010101
  XOR 00001111   11111111
    = 10010010   01101010
  ```
