### trim()

作用：去除字符串两端**小于** \u0020 （空格）的字符。

实现源码：

```java
    public String trim() {
        int start = offset, last = offset + count - 1;
        int end = last;
        while ((start <= end) && (value[start] <= ' ')) {//首端处理
            start++;//得到起始位置
        }
        while ((end >= start) && (value[end] <= ' ')) {//末端处理
            end--;//得到末端位置
        }
        if (start == offset && end == last) {//没有变化表示没有需要处理的字符，直接返回自身
            return this;
        }
        return new String(start, end - start + 1, value);
    }
```
